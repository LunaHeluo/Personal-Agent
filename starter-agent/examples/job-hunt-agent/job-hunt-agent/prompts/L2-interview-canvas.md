# L2 · Interview + Canvas · 任务书

---BEGIN---
你是一名需求研究员。不要先给解决方案。

请根据我的真实经历，追问并整理我目前小的重复性的痛点，例如求职不知道怎么修改简历、不知道怎么安排课程时间等：
- Trigger，当前触发的事件，例如收到jd、收到个人基础简历，收到课表作业信息等
- 当前人工流程，例如根据jd修改简历，根据课程时间安排作业顺序
- 输入材料，例如jd、个人基础简历，课表作业信息等
- Pain，当前痛点的问题，例如不知道怎么修改简历、不知道怎么安排课程时间等
- Done / 可验收结果，例如修改后的简历、安排好的作业顺序
- Human decision，投递简历、开始写作业

然后把结果写入 `starter-agent/docs/interview.md` 和 `starter-agent/docs/product_canvas.md`。

跟做求职示例时，可以参考 `examples/job-hunt-agent/data/`；做自己的 Agent 时，Trigger / Pain 必须来自真实经历。
---END---
