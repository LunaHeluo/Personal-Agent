# L3 · System / Context Split · 任务书

---BEGIN---
请读 `starter-agent/config/prompts/system.md`、`starter-agent/docs/agent.md` 和 `starter-agent/docs/architecture.md`。

修订 system / context 分层说明：
- System 放稳定身份、边界、工具策略。
- Context 放本次 JD、简历、用户偏好、运行状态。
- 不把单次 JD 内容硬编码进 system。
- 不把用户隐私写进长期稳定提示词。
---END---
